Learning Basic HTML & CSS

Reflection on HTML Basics

I’ve learned how to properly structure a webpage using HTML. I now understand how different HTML elements work together to create a well-organized page, and how using semantic tags makes the content more meaningful. This really helped me see how HTML is the foundation of every webpage and why it's so important for both structure and accessibility.

Key Learnings

  • Understanding the purpose of different HTML elements
  • How to use headings correctly to structure content
  1. Learned about HTML validation
  2. Gained practical experience with hands-on coding

For more information, visit Mozilla Developer Network.

Reflection on CSS Basics

In this lab, I really got to see how CSS can make a big difference in how a webpage looks by keeping the design separate from the content. Before this, I didn’t fully appreciate just how much control CSS gives you over things like layout, colors, and fonts.

Key Learnings

  • How to link an external stylesheet to an HTML document
  • The importance of separating content (HTML) from presentation (CSS)
  1. Learned how to apply CSS properties to headings and paragraphs
  2. Gained experience in using class and ID selectors for specific styling

For more information, visit Mozilla Developer Network.
